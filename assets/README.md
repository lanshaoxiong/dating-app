# Assets Folder

This folder contains app assets:

## Required Files:
- `icon.png` - App icon (1024x1024px recommended)
- `splash.png` - Splash screen image
- `adaptive-icon.png` - Android adaptive icon (1024x1024px)
- `favicon.png` - Web favicon (48x48px or larger)

## How to Add:
1. Create or download your app icon
2. Use a tool like https://www.appicon.co/ to generate all required sizes
3. Place the generated files in this folder

## Temporary Solution:
For now, you can use placeholder images or generate simple icons using online tools.
